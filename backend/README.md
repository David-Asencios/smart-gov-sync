Smart-Gov Sync Backend

Comandos locales:

```powershell
npm.cmd install
npm.cmd run seed
npm.cmd start
```

Variables requeridas:

```env
DATABASE_URL=postgresql://usuario:password@host/neondb?sslmode=require&channel_binding=require
JWT_SECRET=clave_segura
PORT=3000
```

Render:

Build Command:

```text
npm install
```

Start Command:

```text
npm start
```
